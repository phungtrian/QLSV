- Chức năng Quản lý sinh viên :
+ Thêm sinh viên : nhập đầu đủ thông tin sinh viên -> chọn Thêm
+ Cập nhật thông tin sinh viên : chọn sinh viên cần sửa trong danh sách -> sửa thông tin trên bản thông tin
-> chọn cập nhật
+ Xóa sinh viên : chọn sinh viên cần Xóa trong danh sách -> chọn Xóa
+ Tra Cứu sinh viên : nhập tông tin tra cứu -> chọn tra cứu
+ Xuất file excel danh sách sinh viên : chọn "Xuất Danh sách sinh viên"

- Chức năng Quản lý Giảng viên :
+ Thêm Giảng viên : nhập đầu đủ thông tin Giảng viên -> chọn Thêm
+ Cập nhật thông tin Giảng viên : chọn Giảng viên cần sửa trong danh sách -> sửa thông tin trên bản thông tin
-> chọn cập nhật
+ Xóa Giảng viên : chọn Giảng viên cần Xóa trong danh sách -> chọn Xóa
+ Tra Cứu Giảng viên : nhập tông tin tra cứu -> chọn tra cứu
+ Xuất file excel danh sách Giảng viên : chọn "Xuất Danh sách Giảng viên"

- Chức năng Quản lý Môn Học :
+ Thêm Môn Học : nhập đầu đủ thông tin Môn Học -> chọn Thêm
+ Cập nhật thông tin Môn Học : chọn Môn Học cần sửa trong danh sách -> sửa thông tin trên bản thông tin
-> chọn cập nhật
+ Xóa Môn Học : chọn Môn Học cần Xóa trong danh sách -> chọn Xóa
+ Tra Cứu Môn Học : nhập tông tin tra cứu -> chọn tra cứu
+ Xuất file excel danh sách Môn Học : chọn "Xuất Danh sách Môn Học"

- Chức năng Quản lý Lớp Học :
+ Thêm Lớp Học : nhập đầu đủ thông tin Lớp Học -> chọn Thêm
+ Cập nhật thông tin Lớp Học : chọn Lớp Học cần sửa trong danh sách -> sửa thông tin trên bản thông tin
-> chọn cập nhật
+ Xóa Lớp Học : chọn Lớp Học cần Xóa trong danh sách -> chọn Xóa
+ Tra Cứu Lớp Học : nhập tông tin tra cứu -> chọn tra cứu
+ Xuất file excel danh sách Lớp Học : chọn "Xuất Danh sách Lớp Học"
+ Xem danh sánh sinh viên (danh sách sinh viên của lớp học): chọn lớp muốn xem trong danh sách > chọn "xem danh sách sinh viên" 
    * Trong form xem danh sách sinh viên*
      - Xem danh sách lớp khác: chọn lớp muốn xem trong mục "chọn lớp" -> click chọn
      - Hủy Môn cho sinh viên(xóa sinh viên ra khỏi lớp): chọn sinh viên muốn hủy môn trong danh sách -> Chọn "Hủy Môn"
      - Nhập điểm: chọn "nhập điểm" (xuấy hiện form nhập điểm của lớp đang chọn hiện tại)
        #chức năng nhập điểm:
	-Chọn lớp : chọn lớp muốn nhập điểm trong mục "Lựa chọn" -> click "chọn"
	-Nhập điểm : chọn sinh viên cần nhập điểm trong danh sách -> nhập điểm trong phần "thông tin chi tiết" -> chọn "nhập"
	-Xuất file excel Bảng điểm sinh viên : chọn "In Bản Điểm"
